== Description ==
This plugin gives you feature to create and add property to showcase and list them on page / post. Plugin is highly customizable real estate, property management, and completely custom listing showcase websites.

Features:
1) Property Creation / Listing.
2) Featured Property listings.
3) Display Properties with shortcodes in pages / widgets.
4) Property Detail View.
5) Property Search Template.
6) Property Images and Video.
7) Google Map Property search filter & much more..

== Installation ==

1. Upload the entire `property-manager` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set permalinks to Post name.
4. Copy shortcode from SETTINGS and paste in page you want to show and select PAGE ATTRIBUTE TEMPLATE to "search property page".
